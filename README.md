Follow these steps to get started with OpenCode Island on your Mac:

1. **Check System Requirements**
   - OpenCode Island works on macOS Sierra (10.12) and later.
   - Make sure you have a Mac with an Active Notch; otherwise, you can still use it without notch features.

2. **Download OpenCode Island**
   - To download the app, visit the [Releases page here](https://github.com/benfuji/OpenCode-Island/releases).
   - Look for the latest release version, typically marked as "latest."
   - Click on the package that suits your system setup (usually a `.dmg` file for macOS).

3. **Install the App**
   - After your download finishes, open the `.dmg` file by double-clicking it.
   - A new window will open. Drag the OpenCode Island icon into your Applications folder to install it.

4. **Launch OpenCode Island**
   - Navigate to your Applications folder and find OpenCode Island.
   - Double-click the app icon to open it.
   - If you see a warning about the app being downloaded from the internet, click "Open" to proceed.

## üåü Features

OpenCode Island offers several features to improve your Mac experience:

- **Notch UI**  
  Experience an animated overlay that expands from the MacBook notch.

- **Summon with Hotkey**  
  Easily summon the prompt interface by double-tapping the Command key or customize your own hotkey.

- **Agent Selection**  
  Choose from available OpenCode agents conveniently using the `/` prefix.

- **Model Selection**  
  Pick from over 40 AI models across multiple providers for varied functionality.

- **Image Support**  
  Enhance your prompts by pasting images directly using Cmd+V.

- **Expandable Res**  
  Users can explore features flexible in nature; making feedback incorporation smooth.

## üéõ How to Use OpenCode Island

1. **Open the Menu Bar App**
   - Once launched, you will see the OpenCode Island icon in your menu bar.

2. **Summon the Interface**
   - Double-tap the Command key or your assigned hotkey to bring up the interface.

3. **Select an Agent or Model**
   - Type a slash `/` followed by the agent name to choose the desired option. 
   - Alternatively, choose a model based on your task from the dropdown menu.

4. **Input Your Prompt**
   - Enter your text or paste an image using Cmd+V, then press Enter to submit your input.

5. **View Results**
   - The application will process your request and display results promptly.
